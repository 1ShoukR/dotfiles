# Dotfiles

My macOS dev environment config.

## What's Included
- `.zshrc` — Oh My Zsh config with Powerlevel10k + plugins
- `.p10k.zsh` — Powerlevel10k theme config
- `.gitconfig` — Git config with delta, rerere, rebase

## Setup on a New Machine

1. Install Oh My Zsh, Powerlevel10k, and external plugins (see .zshrc for list)
2. Clone this repo: `git clone git@github.com:YOUR_USERNAME/dotfiles.git ~/dotfiles`
3. Run: `cd ~/dotfiles && ./install.sh`
4. Restart terminal
